# Groundedness Rubric (v1)

Score 5: All answer claims are explicitly supported by retrieved chunks; citations map cleanly to sources.
Score 4: Minor wording/interpretation, but all substantive claims are supported by retrieved chunks.
Score 3: Mixed; some claims supported, some weakly supported or missing explicit evidence.
Score 2: Mostly unsupported; retrieved context does not justify key claims.
Score 1: Unrelated or fabricated; answer is not grounded in retrieved context.
